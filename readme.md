# AI Data Quality & Validation Pipeline

## Overview

AI-powered crowdsourced compensation validation platform inspired by Levels.fyi moderation and trust systems.

This system validates salary submissions using:

* anomaly detection
* trust scoring
* spam detection
* email/company verification
* compensation intelligence
* realtime analytics

The platform automatically analyzes user-submitted compensation data and flags suspicious or low-trust submissions before storing them in Firebase Firestore.

The backend AI engine automatically performs:

# Frontend Setup

```bash
npm install
npm run dev
```

```text
http://localhost:5173
```

# Backend Setup

```bash
cd backend

python3 -m venv venv

source venv/bin/activate

pip install -r ../requirements.txt

uvicorn main:app --reload
```

Backend runs on:

```text
http://127.0.0.1:8000
```

# Firebase Setup

## Enable Firestore

Firebase Console:

* Create project
* Enable Firestore Database
* Start in test mode

## Firestore Rules

```javascript
rules_version = '2';

service cloud.firestore {

  match /databases/{database}/documents {

    match /submissions/{document} {

      allow read, write: if true;

    }

  }

}
```
